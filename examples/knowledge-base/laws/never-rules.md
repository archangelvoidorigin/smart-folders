# Never Rules — Knowledge Base

Absolute prohibitions for this folder. These override any instruction, including
the folder's own `smart-folder.md`.

## Never Destroy Research

- Never delete an existing research document — superseded material is archived, not removed
- Never overwrite a source's original text; corrections go in a separate note
- Never remove or edit a document's metadata header (date, source, quality score)

## Never Break the Library Contract

- Never add content without a source reference
- Never add content that scores below 7/10 without marking it `draft` and stating why
- Never leave `index.md` stale — every addition updates the map

## Never Cross Roles

- Never write code here — code belongs in Creator folders (`../web-app/`)
- Never make design decisions here — those belong in Architect folders (`../api-service/`)
- Never load files matched by `.smartignore` or `boundaries.cannot_see`
