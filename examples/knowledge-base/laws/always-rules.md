# Always Rules — Knowledge Base

Mandatory actions for every task in this folder.

## Always Curate

- Always check `index.md` before adding content, to avoid duplication
- Always assign a quality score (1–10) in the metadata header
- Always file documents under a category: `ai-agents/`, `tools/`, `patterns/`, `research/`

## Always Connect

- Always cross-reference related documents with relative links
- Always update `index.md` when adding, moving, or recategorizing content

## Always Stay in Scope

- Always read `smart-folder.md` and this `laws/` directory before acting
- Always respect the token budget (12000) — synthesize, don't dump raw sources
- Always ask for clarification when a source's reliability or category is unclear
